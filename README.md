# 🚀 Himanshu Saraswat | Portfolio

A modern, responsive portfolio website built with React, featuring a dark theme with creative animations and effects.

## 🌐 Live Demo

**Portfolio URL:** [https://saraswat2.github.io/portfolio/](https://saraswat2.github.io/portfolio/)

## ✨ Features

- 🎨 **Dark Theme** - Professional navy blue background with red accent colors
- 🎭 **Creative Animations** - Smooth transitions, hover effects, and interactive elements
- 📱 **Fully Responsive** - Works perfectly on all devices
- ⚡ **Fast Performance** - Built with Vite for optimal loading speed
- 🎯 **Multiple Sections**:
  - Home - Introduction with auto-changing titles
  - About - Personal information and interests
  - Skills - Technical and soft skills showcase
  - Projects - Portfolio of completed projects
  - Education - Academic background and certifications
  - Achievements - Awards and recognitions
  - Resume - Downloadable resume
  - Contact - Get in touch

## 🛠️ Technologies Used

- **Frontend Framework:** React 18
- **Build Tool:** Vite
- **Routing:** React Router DOM (HashRouter)
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Styling:** CSS3 with custom animations
- **Deployment:** GitHub Pages

## 📦 Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/saraswat2/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## 🎨 Customization

### Update Personal Information

1. **Home Page** - Edit `src/pages/Home.jsx`
2. **About Section** - Edit `src/pages/About.jsx`
3. **Projects** - Edit `src/pages/Projects.jsx`
4. **Skills** - Edit `src/pages/Skills.jsx`
5. **Education** - Edit `src/pages/Education.jsx`

### Add Your Photos

1. Place your profile photo as `myphotos.png` in the `public` folder
2. Place your resume image as `resume.png` in the `public` folder

### Change Colors

Edit the color variables in `src/App.css`:
- Primary: `#ff6b6b` (Red)
- Secondary: `#ff8e53` (Orange)
- Background: `#1a1a2e` to `#16213e` (Navy Blue gradient)

## 🚀 Deployment

This portfolio is automatically deployed to GitHub Pages using GitHub Actions.

### Deploy Your Own

1. Fork this repository
2. Go to Settings → Pages
3. Select "GitHub Actions" as the source
4. Push changes to the `main` branch
5. Your portfolio will be live at `https://yourusername.github.io/portfolio/`
